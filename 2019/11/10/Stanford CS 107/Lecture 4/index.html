Strings

An array of characters

String Length

How to represent?

  1. Reserve the first byte to store the length: length is limited by size of 1 byte
  2. Terminate every string with a ‘\0’ character: exactly 1 byte used, doesn’t change indices, but traversing the string to get the length

C strings use Option 2

null-terminating character ‘\0’

#include <string.h>

int length = strlen(myStr);// O(n)*********scan the entire string
// should be save if used afterwards

C passes the location of the first character rather than a copy of the whole array

    printf("%s\n", str);

Copying

char str2[5];

int length = strlen(str2); // continue to read other memory

char str1[14];
