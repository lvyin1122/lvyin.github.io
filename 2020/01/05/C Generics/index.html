C Generics - Function Pointers

  • Write C code that works with any data type
  • Pass functions as parameters
  • Functions that accept functions as parameters

Recap:

  1. byte size
  2. convert to char pointer when arithmetic needed
  3. define element size bytes when creating a data structure
  4. Use heap to create new node(stack)
  5. Pass address to create new node
  6. Free the memory space when poping data; need to pass a address variable as well

Bubble Sort

void bubble_sort_int(int *arr, int n){
    while(true){
        bool swapped = false;
        for(int i = 1; i < n; i++){
            if(arr[i-1] > arr[i]){
                swapped = true;
                swap_int(&arr[i-1], &arr[i]);
            }
        }

        if(!swapped){
            return;
        }
    }
}

Generic Bubble Sort

void bubble_sort(void *arr, int n, int elem_size_bytes){
    while(true){
        bool swapped = false;
        for(int i = 1; i < n; i++){
            void *prev_elem = (char *)arr + (i - 1) * elem_size_bytes; //convert to char*
            void *curr_elem = (char *)arr + i * elem_size_bytes; // to fit the generic swap function
            if(){
                swapped = true;
                swap(prev_elem, curr_elem, elem_size_bytes);
            }
        }

        if(!swapped){
            return;
        }
    }
}

Pattern:

  1. Cast the void* argument(s) and set typed pointers equal to them
  2. Derefernce the typed pointer(s) to access the values
  3. Perform the necessary operation

Generic printing

void print_type(void *int_num){
    int *numptr = (int *)int_num; // casting
    int n = *numptr;
    printf("%d\n", n);
}

void print_array(void *arr, int nelems, int elem_size_bytes, void (*print_array_type)(void *int_num)){
    char *ptr = (char*)arr;
    int i;
    for(i = 0; i < nelems; i++){
        ptr += i * elem_size_bytes;
        print_type(arr);
    }
}
